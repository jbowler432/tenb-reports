Patch Cadence Report

This report shows statistics for patching times calculated by exporting the vuln database and calculating the difference between the last_fixed and first_seen dates. Statistics are calculated at the host level first then averaged across all hosts. Different views are provided to show the differences between using medians and means, both at a host and system wide level. Different aggregations are also shown depending on whether severity categories are aligned to CVSS or VPR.
